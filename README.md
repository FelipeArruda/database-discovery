## ⚡️ Installation
```sh
python -m pip install -r requirements.txt
```
* Rename `.env_example` to `.env` and set up your connection to mysql database.
## ⚡️ Used Packages
### Mysql connector
```sh
pip install mysql-connector-python
```
### SQLITE3
```sh
pip install pysqlite3
```
